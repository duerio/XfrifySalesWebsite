Save time transferring files to and from your stores.

DESIGNED FOR RETAILERS

You need to bring back sales and stock information
from your stores,as well as send price updates.

Xfrify lets you automate this process. Tell it what
files to transfer, and when, and Xfrify will do the rest.

It doesn't get any easier.




XFRIFY IS SECURE

Xfrify is cloud based, but can connect over secure VPNs to your stores.

FTP, SFTP and FTPS are supported.
If you don't know what this means, don't worry - it just means Xfrify can handle it.



FILE GROUPS

Setup different file groups.

You can setup file groups - perhaps a sales file group which contains multiple files - item sales file, department sales file and so on.

You might create a new file group for product updates to send to your stores.



LOCATIONS

Setup as many remote stores/locations as you want.

Xfrify allows you to setup location numbers and up to 8 variables, which can be used in filenames for store specific files.

You can setup location groups which contain one or more locations.



SCHEDULES

Schedules are the heart of the system - where you tell the system to run this file group for all locations in this location group, and at a particular frequency. Perhaps once a day? Or once every 3 days?
